Prerequisites

Must have:

  • Ruby
  • Sinatra

Will be less sad if you have:

  • Git (version control)
  • Bundler (dependency management)

Will be happier if you have:

  • Heroku account (free Ruby hosting!)

What is Sinatra?

  • A framework for web apps, like Ruby on Rails, but with a different design philosophy.
  • A Domain Specific Language for writing web apps

Why use Sinatra?

Reach for Sinatra when:

  • Prototyping
  • Making an API
  • No need for CRUD/all the other functionality Rails comes with
  • You want more than Rack but less than Rails

Why not use Sinatra?

  • If your app grows, you may end up rebuilding parts of rails and you may be better served by just switching to rails if that starts to happen.

Who uses Sinatra?

  • Speed of Animals
  • LinkedIn
  • rstat.us did, until we started rebuilding rails.
  • Many many more!

How do I use Sinatra?

  • Bare minimum (examples/hello_world)
  • Routes - pattern matching (examples/route_patterns)
  • Views/Templates - render info from ruby (examples/templates)
  • Routes - HTTP verbs (examples/http_verbs)
  • Deploying to heroku
